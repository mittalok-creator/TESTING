How I Work

STEP 01Understand the number behind the ask

Every dashboard, screen or campaign starts with one question: which decision will this output drive? If it doesn't drive a decision, it doesn't get built.

STEP 02Clean data before clever design

Automated validation and reconciliation first. A beautiful dashboard on wrong data is worse than no dashboard.

STEP 03Automate the repeatable

Anything done twice manually becomes a Power Query step or VBA routine. Humans should review, not copy-paste.

STEP 04Design for the reader, not the builder

Whether it's an MIS pack or a mobile screen — hierarchy, whitespace and one clear next action. Clarity converts.
